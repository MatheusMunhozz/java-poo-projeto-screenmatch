# 🎬 Java POO - Screenmatch

## Sobre o projeto

Este é o projeto **Screenmatch** desenvolvido durante meus estudos de **Programação Orientada a Objetos (POO)** em Java na Alura. O projeto simula um sistema de cadastro e avaliação de filmes.

## Funcionalidades

- Cadastro de filmes (nome, ano de lançamento, duração)
- Sistema de avaliação de filmes (notas de 0 a 10)
- Cálculo da média das avaliações
- Exibição de ficha técnica
